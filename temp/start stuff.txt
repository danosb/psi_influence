for Mysql server:
You can use the following command to start the server:
	sudo /usr/local/mysql/support-files/mysql.server start

And you can use the following command to stop the server:
	sudo /usr/local/mysql/support-files/mysql.server stop

Once the server is running, you can use the mysql command-line tool to connect to the server and execute SQL queries. 
	mysql -h localhost:3306 -u root -p Password1!
